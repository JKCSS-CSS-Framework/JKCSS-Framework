Some Color Variations

use Following color styles for Variations

for text color Variations

Light: <div class="text-COLOR_NAME-light-NUMBER"> ... </div>
Dark: <div class="text-COLOR_NAME-dark-NUMBER"> ... </div>

for Background color Variations

Light: <div class="bg-COLOR_NAME-light-NUMBER"> ... </div>
Dark: <div class="bg-COLOR_NAME-dark-NUMBER"> ... </div>

Inorder to use Color Variations Follow the following Steps

1. in Abouve Example COLOR_NAME should be change to color name

2. in Abouve Example NUMBER should be change to between 1 - 9
